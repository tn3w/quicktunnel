STEP 01

COMMAND EXPLAINED
ssh
↓
OpenSSH client

Pre-installed on Linux, macOS, Windows 10+. No download needed.

-oStrictHostKeyChecking=no
↓
Skip host prompt

Suppresses the "Are you sure?" fingerprint prompt on first connect. Safe for ephemeral tunnels.

-N
↓
No remote command

Opens the tunnel only — no shell session, no login, nothing runs on the server.

-R 80:localhost:3000
↓
Remote port forward

Binds port 80 on the server to your local port. Traffic in → your machine out. That's it.

t.tn3w.dev
↓
Tunnel server

The public relay. Assigns you a URL and forwards HTTP traffic to your tunnel.

🔍 This command opens an outbound SSH connection from your machine — just like visiting a website. It does not give the server access to your system, files, or any other ports.
